# ai-inference-vision-bot

## How To Use
   
1. Git clone or download the repository:
  ```
  git clone https://github.com/cpoc-internal/ai-inference-vision-bot
   cd ai-inference-vision-bot
  ```
2. Install required libraries for this AI chatbot
  ```
  chmod +x prep-server.sh
  ./prep-server.sh
  ```
3. 
  - **after the server is ready, please run the followwing command
  ```
     docker-compose up --build
  ```
